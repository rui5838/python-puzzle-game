# Block Blast Clone

Pythonで開発したパズルゲーム「Block Blast」のクローンアプリケーションです。
既存のモバイルゲームのビジュアルスタイルを意識したUI設計と、快適なプレイ体験のためのロジックを実装しています。

## 開発のポイント

* **モバイルライクなUI実装**: 既存ゲームの視覚的なスタイルに近づけるため、UIのレイアウトやデザインを細部まで調整し、直感的な操作感を実現しました。
* **Perfect Function (最適配置ロジック)**: プレイヤーの操作性を向上させるための独自のブロック配置判定システムを構築し、ゲームのコアロジックに組み込んでいます。
* **スタンドアロン化**: PyInstallerを使用し、Python環境が構築されていないWindows PCでも即座に起動できる実行ファイル（.exe）としてパッケージングを行いました。

## 使用技術

* Python
* PyInstaller (実行ファイル化)

## アプリの起動方法

環境構築は不要です。右側の「Releases」から `block_blast.exe` をダウンロードし、ダブルクリックするだけでゲームを開始できます。
